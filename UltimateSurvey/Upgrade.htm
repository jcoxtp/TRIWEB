Ultimate Survey Advanced Edition - v6.7.4 Upgrade Documentation
by Ultimate Software Designs ©2003

If you want to upgrade to Ultimate Survey Advanced Edition Version 6.7.4 and keep all of your current data, please follow the instructions in this file.  If you want to install version 6.7.4 with a brand new database, you can follow the instructions in Install.htm.  These instructions are for users who are currently running Ultimate Survey 6.6.x. or 6.7.x  

If you have already upgraded to version 6.7, 6.7.1, 6.7.2, or 6.7.3 please skip to step 4

1)  Backup your Ultimate Survey directory and backup your database.  We cannot guarantee support for upgrades if you do not do this.  You can perfom this upgrade against the current database or the copy.  If you do not do it against the copy, any logged in users may experience delays and other problems. 

2)  If you have not already done so, update your database to correspond to Ultimate Survey Advanced Edition v6.6.1.  You can do this by following the upgrade instructions in the files available in our customer area at http://www.ultimateapps.com/support/members .  You do not need the ASP files from the previous versions

3)  Update your database to Version 6.7.   

  • SQL Server Users - Open query analyzer pointing to your Ultimate Survey database.  Open the file "Database/databaseUpdate.sql" and run it.
  • MSAccess Users
    • Add the following fields to existing tables:
      Table Name Field Name Field Type
      usd_answers orderByID number
      usd_matrixSets fieldLength number
      usd_surveyItem pipedItemID1 number
      usd_surveyItem pipedItemID2 number
      usd_surveyItem pipedItemID3 number
      usd_surveyItem numberRows number
      usd_surveyItem numberColumns number
      usd_surveyItem variableName text
      usd_surveySettings preventConcurrentLogin number
      usd_surveySettings cookieName text
      usd_surveyUser userGUID text

4) Make database changes for Version 6.7.4.

  • SQL Server Users - Open query analyzer pointing to your Ultimate Survey database.  Open the file "Database/databaseUpdate-6.7.4.sql" and run it.
  • MSAccess Users
    • Add the following fields to existing tables:
      Table Name Field Name Field Type
      usd_matrixAnswers orderByID number
      usd_matrixCategories orderByID number

5)  Update your Ultimate Survey directory with files

  • Copy the files contained in the zip to your Ultimate Survey directory.  If you are using the same database as you did for previous versions, do not overwrite "Include/SurveyConnection_inc.asp".  If you are using a different database, please update your connection string in this file.
