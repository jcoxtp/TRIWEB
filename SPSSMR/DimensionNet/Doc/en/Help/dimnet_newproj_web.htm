Modifying Web Service Settings

You may also need to alter the default configuration of SyncWebService. If this service will run on a server that does not also host Web interviews, the web.config file should be modified so that the ShouldHostTransferService setting is set to True. Without this change, the case data sent to the server will not be merged into the database.

For more information, see "Modifying the Default Web Service Settings for Remote Manager" in the mrInterview Installation Instructions and Configuration Notes.
