DSCs Supplied with the Dimensions Data Model

DSCDescriptionInternal Name
ADO DSCReads case data and metadata from ADO files.mrAdoDsc
Delimited Text DSCReads and writes tab-delimited, variable length, case data records to a text file.mrCsvDsc
Dimensions Data File CDSCReads and writes case data to a Dimensions file-based database. The Dimensions Data File CDSC has all the features of the Relational MR Database (RDB) CDSC, but does not require a separate relational database management system. Dimensions Files therefore provide a simpler and more portable format for the storage of "offline" mrInterview data.mrDataFileDsc
In2data DSCReads In2data case data and metadata.mrI2dDsc
Log DSCReads standard Dimensions log files.mrLogDsc
QDI/DRS DSCReads case data and metadata in the standard Quancept QDI/DRS format.mrQdiDrsDsc
Quantum DSC Reads and writes case data in a Quantum-format ASCII data file. Writes metadata in the form of a basic Quantum specification.mrPunchDsc.
Quanvert DSCReads metadata and case data in a Quanvert database.mrQvDsc
Relational MR (RDB) Database CDSCReads and writes case data to an SPSS Dimensions relational database. The RDB DSC currently supports SQL Server databases only. This is the CDSC that mrInterview uses to write response data.mrRdbDsc2
SPSS SAV DSCReads and writes metadata and case data in an SPSS .sav file.mrSavDsc
SurveyCraft DSCReads SurveyCraft case data and metadata.mrScDsc
Triple-S DSCReads and writes case data in a fixed-format or comma-separated text file that meets the version 1.2 or 2.0 Triple-S standard. Reads and writes metadata in a version 1.2 or 2.0 Triple-S XML file.mrTripleSDsc
XML CDSCReads and writes case data to an XML file. Typically, you use this CDSC when you want to demonstrate or test the tabulation of small volumes of case data. To tabulate "offline" production data, use the Dimensions Data File CDSC.mrXmlDsc

For detailed information about these DSCs and how they interpret the underlying data, see the Dimensions Development Library (DDL) documentation.
