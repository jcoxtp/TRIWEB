Logging In as a Different User

If you are logged in to DimensionNet under one user name and you need to do some work using a different user name, there is no need to close your existing session. Instead, you can return to the login page and enter a different user name and password. This is a handy time-saver for administrators who also have another DimensionNet account for non-administration work.

To Log In as a Different User

► Click the Logout icon in the DimensionNet menu bar.

► On the Login page, enter a user name and password and click Login.
