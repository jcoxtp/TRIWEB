Welcome to DimensionNet

DimensionNet is an easy-to-use tool for running a variety of Dimensions Web activities on a project of your choice. It also provides the following management and maintenance facilities:

• EditProject for setting and changing project properties.

• Manage Users for maintaining user and group accounts.

• Manage Access for determining which user groups have access to a project and who can use certain features within an activity.

• Files for copying project files between your working folder and the shared project folder and between DimensionNet and your computer.

• Usage Reports for viewing DimensionNet usage reports.

• Manage Logs for viewing log files for DimensionNet and the activities that run inside it.
