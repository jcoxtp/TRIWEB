Setting Up a Mobile Interviewing Project

This section describes how to set up a project for mobile interviewing. In order for a project to be considered a mobile project, it must have the IsMobile property set to True in the Distributed Property Management (DPM) system. If this property is not set, the project is not synchronized to remote devices.

To Set the IsMobile Property:

► Select the project in the project list and click the Edit Project Properties button.

► On the Properties tab, scroll down to the bottom of the list of properties and click Click here to add property. A new property with placeholder values is added to the list.

► Select NewProperty and replace it with IsMobile.

► Select SomeValue and replace it with True.

► Select Boolean from the Type menu.
