Setting DimensionNet Options

DimensionNet options control the following aspects of DimensionNet behavior:

Project list. You can control which types of projects you see in your project list and the order in which they are shown. Folders are always sorted in alphabetical order, but within folders projects can be sorted by name or status, and you can choose to exclude projects with certain statuses from the list.

Activities list. You can sort activities by name or type, and can suppress their descriptions.

Language. The text in the DimensionNet GUI is normally displayed in English, but translations into other languages may be available if you have the Dimensions Language Pack installed. If your browser's language is set to a language that is available in DimensionNet, then your DimensionNet language will be set to match the browser language.

If you have your browser set to work in one language but want to run DimensionNet in a different language, you may set the language within DimensionNet. For example, if your browser runs in Japanese, then DimensionNet will automatically run in Japanese too. If your browser runs in English and you want to run DimensionNet in Japanese, set the language to Japanese in DimensionNet.

Browser favorite. You can add DimensionNet to your browser's Favorites.

To Control How DimensionNet Displays Projects and Activities:

► Choose:

 Options
  Preferences

This opens the Project Options page.

► Choose the Project and Activity options you want.

► In "Select your preferred language", choose the language in which you want the DimensionNet GUI texts such a menu names and error messages to be displayed. (The availability of translations depends on which version of the Dimensions Language Pack is installed. If DimensionNet does not display texts in the language you have chosen, check this with your DimensionNet administrator.)

► Uncheck Show Activity Descriptions if you do not want to see descriptions for each activity.

► If you want to add DimensionNet to your browser's Favorites list, do the following:

• Click Add to Favorites. This opens the standard browser Add Favorite dialog box.

• Choose where the link should appear in the favorites list and click OK.

► Click Apply Changes.
