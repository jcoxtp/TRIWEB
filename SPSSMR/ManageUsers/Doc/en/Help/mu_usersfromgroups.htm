Removing Users from Roles

Either:

► Expand the Roles section of the tree and select the role from which you want to delete users.

A list of current role members appears on the right-hand side of the page.

► Click unassign on the line for the user you want to remove, or click unassign user(s) to remove a set of users (you will be prompted to choose the user names from a list).

► Confirm your request when prompted to do so.

Or:

► Expand the Users section of the tree and select the user you want to remove from a role.

A list of the roles that the user belongs to appears on the right-hand side of the page.

► Click unassign on the line for the role from which the user should be removed, or click unassign roles(s) to remove a set of roles (you will be prompted to choose the role names from a list).
