Creating User Property Groups

► Click the Edit user properties icon in the toolbar.

This opens the Edit User Properties dialog box.

► In "Define a group of user properties for", select the role to which the property group refers.

► In the Property Groups frame, click the Add property group icon in the title bar or click <Click here to add new>.

This opens a new page in the dialog box.

► In Group Name, type a name for the group you are creating. The group name for interviewer qualifications must be InterviewerQualifications.

► In Group Description, type a description of the group's function.

► Click Create.

You are returned to the previous page of the dialog box, which now provides facilities for adding properties to the group.
