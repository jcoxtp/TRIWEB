Removing Users

Either:

► In the User Roles tree, click Users.

A list of current user names appears on the right of the screen.

► Click remove on the line for the user you want to remove.

► Confirm the request when prompted to do so.

Or:

► In the User Roles tree, expand the Users branch and select the user you want to remove.

► Click remove user in the frame on the right of the page.

► Confirm the request when prompted to do so.
