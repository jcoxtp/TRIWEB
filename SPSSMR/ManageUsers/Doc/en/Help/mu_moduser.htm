Modifying User Accounts

You can change the details stored for a user account, the properties assigned to that user, and the roles of which that user is a member.

If you are the DPM administrator, you can change a user's password by modifying the user's account details. If you want to be able to change the password for the DPM administrator account itself, you will need to edit that account and select the Change Password check box, and then change the password.

► In the User Roles tree, expand the Users line.

► Select a user name.

Information about the user name appears on the right-hand side of the page.

► To change the user's account details:

• Click modify user.

The SPSS Manage Users dialog box displays the current account settings.

► Make your changes (you cannot change the user name).

If user properties have been defined in DimensionNet, the User Properties button is displayed. To assign new properties to this user or change existing assignments, click this button to open the Assign User Properties dialog box, and then make your changes and click OK to return to the user details dialog box.

► Click OK.

► To assign the user to one or more roles:

• Click assign role(s).

This opens the SPSS Manage Users dialog box.

• Select the roles you want to add the user to and click OK.
