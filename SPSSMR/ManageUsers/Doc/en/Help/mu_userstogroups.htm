Assigning Users to Roles

You can assign users to roles when you add the user to DimensionNet, or as a separate task at a later date.

Note: If you want mrInterview to take interviewer qualifications into account when assigning calls to interviewers, you must make interviewers members of the CATIinterviewer role.

Either:

► Expand the Roles section of the tree and select the role to which you want to add users.

A list of current role members appears on the right-hand side of the page.

► Click assign user(s).

This opens the SPSS Manage Users dialog box.

► Select the user names to be added to the role and click OK.

Or:

► Expand the Users section of the tree and select the user you want to add to a role.

A list of the roles that the user belongs to appears on the right-hand side of the page.

► Click assign role(s).

This opens the SPSS Manage Users dialog box.

► Select the roles to which the user should be added and click OK.
