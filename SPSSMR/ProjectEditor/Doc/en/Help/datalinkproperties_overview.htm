Data Link Properties Dialog Box

When you access data using the Dimensions Data Model, there are a number of connection properties that need to be specified. The Data Link Properties dialog box makes it easy to set up the connection properties.

The Data Link Properties dialog box has three tabs--Connection, Advanced, and All.
