License Report

The license report shows the number of licenses available and the number of licenses being used for a chosen activity in a chosen period. The activities you can report on, either individually or all together, are:

• Phone Participants

• mrInterview

• mrTables

• Reporter

The report can cover just the current month or the previous month, six months, or 12 months, or all these periods.

The report for the current month shows three sets of figures: the current number of licenses and users, the maximum number of licenses and users this month, and the maximum number of licenses and users ever. The other monthly reports show only the maximum number of licenses and users in the period. The "all periods" report combines these two approaches, showing detailed information for the current month and summary information for the previous months.

To create a license report:

► In Time Span, select the period that the reports are to cover.

► In Activities, select the activity whose licenses you want to check, or choose All activities for a complete report.

► In Servers, choose the servers to include in the report. You may choose a single server or all servers.

► Click View.

The report is displayed on your screen. Click Download Table if you want to download it onto your computer.A standard Windows File Open dialog box is displayed with options to open or save the file.

► Click Open to open the file or Save to save it. The default filename is Concurrent_Report_date_time.xml; for example, Concurrent_Report_20071120_95218.
