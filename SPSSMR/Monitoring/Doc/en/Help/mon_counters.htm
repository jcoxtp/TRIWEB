Counter Report

Note: This report is not available by default. Contact your Dimensions administrator if you would like to see it

The counter report tells you how many people have used various DimensionNet activities during a particular period. The counters you can report on, either individually or all together, are:

• Total number of interviews

• Total number of active interviews

• Maximum number of concurrent interviews

• Build activity—current users

• Build activity—maximum number of users

• CATI interviewer—current users (for Phone Participants)

• CATI interviewer—maximum number of users

• Tabulation activity—current users (for mrTables)

• Tabulation activity—maximum number of users

The report can cover just the current month or the previous month, six months, or 12 months, or all these periods.

The report for the current month shows three sets of figures: the current number of users, the maximum number of users this month, and the maximum number of users ever working concurrently. The other monthly reports show only the maximum number of users in the period. The "all periods" report combines these two approaches, showing detailed information for the current month and summary information for the previous months.

To create a counters report:

► In Time Span, select the period that the reports are to cover.

► In Counters, select the counter you want to see, or choose All counters for a complete report.

► In Servers, choose the servers to include in the report. You may choose a single server or all servers.

► Click View.

The report is displayed on your screen. Click Download Table is you want to download the report onto your computer.

A standard Windows File Open dialog box is displayed with options to open or save the file.

► Click Open to open the file or Save to save it. The default filename is Usage_Report_date_time.xml; for example, Usage_Report_20071120_95218.
