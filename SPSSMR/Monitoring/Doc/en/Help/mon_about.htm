Usage Report

Usage Report provides you with license and general usage information for a variety of Dimensions activities.

• The license report shows the number of licenses available and the number of licenses being used for a chosen activity in a chosen period.

• The counter report shows the number of people that have used for an activity in a chosen period. (This report is not available by default. Contact your Dimensions administrator if you would like to see it.)

Both types of reports can be downloaded as .xml files for use outside DimensionNet.
