Deleting Private Files

You can delete files that exist in your user folder and that are marked as Private. You cannot delete files that are marked as Checked In or Checked Out.

To Delete Private Files:

► In the list of filenames, select the file you want to delete.

► From the Files menu, choose:

 Actions
  Delete

► Confirm your request when prompted to do so.
