Assigning Users or Roles to a Project

Do one of the following. Either:

► In the DPM tree frame, expand Users if you want to assign individual users to a project, or Roles to assign all members of a role to a project.

► Select the user or role you want to assign.

This displays a frame on the right of the page listing the projects and activities to which this user or role is currently assigned.

► Click assign to projects.

This opens the SPSS Manage Access dialog box.

► Choose the projects to which this user or role should be assigned and click OK.

Or:

► In the DPM tree frame, expand Projects.

► Select the project name entry.

This displays a frame on the right of the page listing the users and roles currently assigned to this project.

► Click assign user(s) or assign role(s) as appropriate.

This opens the SPSS Manage Access dialog box. The dialog box lists user names or role names according to the option you chose.

► Choose the users or roles you want to assign to this project and click OK.
