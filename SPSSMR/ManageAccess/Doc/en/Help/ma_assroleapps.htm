Assigning Users or Roles to Activities

Do one of the following. Either:

► In the DPM tree frame, expand Users if you want to assign an individual user to an activity, or expand Roles if you want to assign all members of a role to an activity.

► Select the user or role you want to assign.

This displays a frame on the right of the page listing the projects and activities to which this user or role is currently assigned.

► Click assign to activities.

This opens the SPSS Manage Access dialog box.

► Choose the activities to which this user or role should be assigned and click OK.

If any of the chosen activities has activity features, a dialog box is displayed listing the features for those activities.

► Select the features that you want this user or members of this role to be able to use and click OK.

Or:

► In the DPM tree frame, expand Activities.

► Select the activity.

This displays a frame on the right of the page listing the users and roles currently assigned to this activity.

► Click assign user(s) or assign role(s).

This opens the SPSS Manage Access dialog box. The dialog box contains user names or role names depending on the option you chose.

► Choose the users or roles you want to assign to this activity and click OK.

If the activity has activity features, this displays a dialog box listing those features.

► Select the features to be assigned to this role and click OK.
