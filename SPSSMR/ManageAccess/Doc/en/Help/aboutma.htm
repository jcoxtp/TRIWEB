Manage Access

You use Manage Access to specify access rights to projects, case data, and activity features for different users and user groups (roles).

All information about projects, activity features, and access rights is stored as properties in the Distributed Property Management system (DPM). Manage Access displays this information using a tree structure similar to that of Windows Explorer and provides a flexible method of changing and adding information.

When you finish using Manage Access, click the Home icon on the menu bar.

Note: If you want to use Manage Access, you must be authorized in DimensionNet to use it and must be a member of the role named in the HKEY_LOCAL_MACHINE\SOFTWARE\SPSS\MRPropertyManagement\2.0\Security\AdministratorRole registry key (which is the DPMAdmins role by default). For further details, open the Dimensions Development Library documentation and search for the topic entitled "Who Can Run Manage Users and Manage Access."
