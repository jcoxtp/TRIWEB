Removing Users or Roles from Activity Features

► In the DPM tree frame, expand Users to remove users from Features, or Roles to remove all members of a role from Features.

► Expand the entry for the user or role you want to remove.

► Expand Activities.

► Expand an activity and select its Features entry.

This displays a frame on the right of the page listing the activity's features that have been set for this user or role.

► Either click unassign next to a single feature or click unassign activities features and choose one or more features from a list.

► Confirm your request when prompted to do so.
