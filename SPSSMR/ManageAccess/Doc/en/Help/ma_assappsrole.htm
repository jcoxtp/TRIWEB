Assigning Activities to a User or Role

► In the DPM tree frame, expand Users to assign activities to individual users, or Roles to assign activities to all members of a role.

► Expand the user or role you want to assign.

► Select Activities.

This displays a frame on the right of the page listing the activities that the chosen user or role may use.

► Click assign to activities.

This opens the SPSS Manage Access dialog box.

► Select the activities to assign the current user or role to and click OK.

If any of the chosen activities has activity features, a dialog box is displayed listing the features for those activities.

► Select the features that you want this user or members of this role to be able to use and click OK.
