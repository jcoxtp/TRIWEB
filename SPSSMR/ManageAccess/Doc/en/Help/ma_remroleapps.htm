Unassigning Users or Roles from Activities

Do one of the following. Either:

► In the DPM tree frame, expand Users if you want to unassign an individual user from an activity, or expand Roles if you want to unassign all members of a role from an activity.

► Select the user or role you want to unassign.

This displays a frame on the right of the page listing the projects and activities to which this user or role is currently assigned.

► Either click the unassign link for a single activity or click unassign activities and choose one or more activities from a list.

Or:

► In the DPM tree frame, expand Users if you want to unassign an individual user from an activity, or expand Roles if you want to unassign all members of a role from an activity.

► Expand the user or role you want to unassign.

► Select Activities.

This displays a frame on the right of the page listing the activities assigned to this user or role.

► Either click the unassign link for a single activity or click unassign activities and choose one or more activities from a list.

► Confirm or cancel your request when prompted to do so.

Or:

► In the DPM tree frame, expand Activities.

► Select the activity.

This displays a frame on the right of the page listing the users and roles assigned to this activity.

► Either click the unassign link for the user or role you want to remove, or click unassign user(s) or unassign role(s) and choose one or more users or roles from a list.

► Confirm or cancel your action when prompted to do so.

Or:

► In the DPM tree frame, expand Activities.

► Expand the activity.

► Select Users if you want to unassign an individual user from an activity, or Roles if you want to unassign all members of a role from an activity.

This displays a frame on the right of the page listing the users or roles assigned to this activity.

► Select the user or role you want to unassign from this activity.

► Click unassign user from activity or unassign role from activity as appropriate.
