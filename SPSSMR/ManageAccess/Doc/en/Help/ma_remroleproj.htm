Unassigning Users or Roles from Projects

Do one of the following. Either:

► In the DPM tree frame, expand Users if you want to unassign individual users from a project, or Roles to unassign all members of a role from a project.

► Select the user or role you want to unassign.

This displays a frame on the right of the page listing the projects and activities to which this user or role is currently assigned.

► Either click the unassign link for a single project or click unassign projects and choose one or more projects from a list.

Or:

► In the DPM tree frame, expand Users if you want to unassign individual users from a project, or Roles to unassign all members of a role from a project.

► Expand the user or role you want to unassign.

► Expand Projects and select a project.

A new frame appears on the right of the page.

► Click unassign role from project.

► Confirm or cancel your request when prompted to do so.

Or:

► In the DPM tree frame, expand Users if you want to unassign individual users from a project, or Roles to unassign all members of a role from a project.

► Expand the user or role you want to unassign.

► Select Projects.

This displays a frame appears on the right of the page listing the projects currently assigned to this user or role.

► Either click the unassign link for a single project or click unassign projects and choose one or more projects from a list.

► Confirm or cancel your request when prompted to do so.

Or:

► In the DPM tree frame, expand Projects.

► Expand the project.

► Expand Users if you want to unassign individual users from a project, or Roles to unassign all members of a role from a project.

► Select the entry for the user or role you want to unassign from the current project.

► Click unassign user from project or unassign role from project as appropriate.

► Confirm or cancel your request when prompted to do so.

Or:

► In the DPM tree frame, expand Projects.

► Expand the project.

► Click Users or Roles.

This displays a frame on the right of the page listing the users and roles currently assigned to this project.

► Either click the unassign link for a single project, or click unassign users or unassign roles and choose one or more users or roles from a list.

► Confirm or cancel your request when prompted to do so.
