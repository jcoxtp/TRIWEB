Assigning Users or Roles to Activity Features

Activity Features allow you to restrict access to key features within an activity to certain users only. For example, in Project Editor, you might wish to restrict the ability to delete projects to project managers only. Each activity and activity feature has two names: the one by which it is known internally by DimensionNet and the one that is displayed. To Assign Users or Roles to Features

► In the DPM tree frame, expand Users to assign users to Features, or Roles to assign all members of a role to Features.

► Expand the user or role you want to assign.

► Expand Activities.

► Expand an activity and select its Features entry.

This displays a frame on the right of the page listing the activity's features that have been set for the chosen user or role.

► Click assign to activities features.

This opens the SPSS Manage Access dialog box.

► Select the features that you want to make available to the current user or role and click OK.
