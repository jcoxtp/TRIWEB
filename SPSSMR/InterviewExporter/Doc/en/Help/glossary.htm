Glossary

Superversion. A combination of two or more versions of the metadata. The versions are combined to form a superset, although when there is a conflict between, for example, a text in one or more of the versions, the more recent versions generally take precedence over the older versions. However, you can sometimes specify the order of precedence. You typically use a superversion when you want to export or analyze response data that has been collected using more than one version of the questionnaire.Version. Refers to changes to the content of a project. As the project changes (with new questions or categories in a tracking study, for example), each set of such changes will generate a new version. Each generated version is stored in the metadata and you can choose the version or versions you want to use.
