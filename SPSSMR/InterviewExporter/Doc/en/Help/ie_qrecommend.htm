Recommended Procedure for Exporting mrInterview Data to Quantum

The following procedure is recommended for exporting data in Quantum format.

► In DimensionNet, run the Launch activity to activate the project.

This ensures that the .mdd file contains up to date information about the questions in the questionnaire.

► Run MDM2Quantum on the .mdd file you have just updated.

► Reactivate the questionnaire.

The version of the .mdd file in the Shared project folder will now contain card and column information.

► In DimensionNet, run Export Data to export the data.
